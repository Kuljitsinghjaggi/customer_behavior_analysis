# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering business insights through dashboards and reports. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and visualization using Power BI.

The goal of this project is to extract meaningful insights and present them in a clear, actionable format for decision-making.

---

## 📂 Dataset

* Source: *(customer_behavior.csv)*
* Format: CSV / Excel / Database
* Description: The dataset contains information related to behavior of shopping of male , female , adults, childrens.

* Removed duplicates

* Imported cleaned data into database

* Built interactive dashboard

### 6. Reporting

* Highlighted key trends and business recommendations

### 7. Presentation
